This is an example site built with the **Shizuka** Hugo theme.

Shizuka is a minimal, clean blog theme with dark mode, reading time, archive support, and full i18n.

## Features

- Clean, distraction-free reading experience
- Dark mode (auto-detect + manual toggle)
- Reading time estimate
- Archive page grouped by year
- RSS feed
- Tag and category taxonomy
- Table of contents support
- Responsive layout
- i18n support (English, Italian)
