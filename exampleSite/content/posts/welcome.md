Welcome to **Shizuka** — a minimal Hugo blog theme inspired by the Japanese aesthetic of *shibui*: quiet elegance, nothing in excess.

## What is Shizuka?

Shizuka (静か) means "quiet" or "calm" in Japanese. This theme is built around the idea that a good reading experience should get out of the way of the words.

No sidebars. No widget clutter. Just clean typography and your content.

## Features at a glance

- **Dark mode** — auto-detected from system preference, with a manual toggle
- **Reading time** — estimated reading time on every post
- **Archive** — a full chronological index of all posts, grouped by year
- **i18n** — English and Italian built-in, easily extensible
- **RSS** — automatic feed generation
- **TOC** — optional table of contents per post

## Getting started

Install the theme and configure `hugo.toml` with your own title, description, and social links. Then start writing.

```bash
git submodule add https://github.com/carmelolg/shizuka themes/shizuka
```

That's it. Start the server and write.
